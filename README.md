
# ReactJS + Node + Redux - A management app inspired by Trello

## Requirements

* **nodejs >= v14.7**
* **npm >= v6.14.7**

How to run the project:
```
$ git clone <project_url> 
$ cd project_managment
$ npm install 
$ npm start
```
